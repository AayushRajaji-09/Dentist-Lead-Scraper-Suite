# 🚀 Antigravity Dentist Lead Scraper Suite v2.0

An automated, customizable B2B lead generation and profiling suite across 10+ major Indian cities (`Ahmedabad`, `Mumbai`, `Surat`, `Pune`, `Bangalore`, `Delhi`, `Hyderabad`, `Kolkata`, `Chennai`, `Jaipur`). Equipped with an interactive GUI (`tkinter`), deep Google Maps extraction (`Ratings`, `Reviews`, `Ad Status`, `Website` checks), auto-discovery vectors, and automated Git telemetry.

---

## 📦 What's Inside

- **`dentist_lead_suite_gui.py`** — Interactive graphical application (`tkinter`) for non-tech users.
- **`free_dentist_scraper.py`** — Standalone multi-area command-line scraper script.
- **`run_logger.py`** — Automated Git telemetry and metrics tracking (`run_history.json`, `run_log.csv`).
- **`HOW_TO_USE.md`** — Complete step-by-step user manual, strategic agency outreach guide, and Excel data profiling schema.
- **`requirements.txt`** — Python package dependencies (`playwright`, `pandas`, `openpyxl`).

---

## 🟢 Quickstart (1 Line)

1. Clone this repository and install dependencies:
   ```powershell
   pip install -r requirements.txt
   playwright install chromium
   ```
2. Launch the GUI Suite:
   ```powershell
   python dentist_lead_suite_gui.py
   ```

*(For detailed column definitions, strategic agency pitching tips, and troubleshooting, read **[`HOW_TO_USE.md`](HOW_TO_USE.md)**.)*
